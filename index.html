The JavaScript Basics!

What is JavaScript?

  • JavaScript 'thinks' in terms of "things/objects" in JS they are called Objects.
  • This DOM or Document Object Model is the basic foundation of the entire JavaScript language.
  • Objects contain objects: a pc has a browser which has a webpage which has a document.
  • Objects have properties: a webpage (Object) has a background (Object) color (Property), the color is a property.
  • JavaScript causes Actions to happen on or with Objects: These actions are called Methods.
  • Methods can be referred to as Functions.
  • Functions are Methods that you create/setup to perform a Task.
  • Events: Normally user action like moving mouse/clicking
  • Event Handlers: Actions to take when an Event occurs, which calls JavaScript.

Events/Event Handlers

  • Events are often users visiting/mouse moving/clicks which cause the Event Handler to react.
  • Event Listener: The mechanism for detecting events and triggering a response.
  • Event Handler: The function or code that is executed in response to the event.
  • An example would be a user clicks on a link (Event) and the Event Handler takes you to the specified link.
  • Event Handlers use variables and methods to perform the tasks they've been told to do.
  • Move Mouse Over To Cause Event. HTML has the Event Listener. JS function is the Event Handler.
  • Event: Mouse Movement | Events Listeners: onMouseOver & onMouseOut | Event Handlers: makeGreen() & makeYellow()
    • // This is the event handler function
    function handleClick() {
    console.log('The element was clicked!');
    }

    // Adding the event listener to an element
    document.getElementById('myElement').addEventListener('click', handleClick);
  • Objects can be presented as either methods 'document.getElementById()' or properties 'document.URL' with dots between.
  • Events are not always user dependant, for example the 'onLoad' Event Handler
  • Additional Event Listeners: onBlur, onFocus, onSelect, & onChange mainly used with HTML forms

Statements

  • Statements tell Scripts if some thing is True, Present, or Not.
  • Common Statements: If, For, Else, and Else If.
    • if ( x > 1) {}: if x is greater than 1. Run Code.
    if !( x > 1) {}: if x is not greater than 1. Run Code
    for (let x = 0; (x < 10); x++) {}: While x less than 10. Run Code
    if ((x == 1) && (y == 2)) {} else if (x == y || x <= y) {} else {}: if x is equal to 1 and y is equal 2. Run Code. Else if x equals y or x less than equal to y. Run Other Code. Else. Code Other Code.
  • The 'for' Statement is used to run a Loop

Functions & Methods

  • Functions and Methods are basically the same thing
  • Functions differ slightly by including event handlers, variables, and statements.
    • document.write() <= Method
    openPopup(x) {} <= Function
  • Fuctions are not loaded until called or user performs an action.

Errors

  • Two Types of Errors: Syntax & Runtime
  • Syntax: Code is Wrong, Typo or Variable Undefined.
  • Runtime: Script is Broken.
  • Errors in browsers often give you line. Fix errors from the top down.
